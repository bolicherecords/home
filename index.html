Proyectos

ZSMO

ZSMO es un proyecto de mejora continua para la Sociedad Madariaga Olivares, una PyME del rubro de la exportación e importación de calzados en la ciudad de Arica. El proyecto busca escalar su sistema desarrollado en lenguaje PHP, potenciado por Yii Framework y una base de datos MySQL, cubriendo todas las aristas de la empresa y además explotar la información mediante la creación y mejoras de reportes dinámicos.

RapaCargo

RapaCargo es un proyecto de mejora continua para la Sociedad Rapanui Cargo, una PyME del rubro de la logística de paquetes entre Santiago e Isla de Pascua. El proyecto busca escalar su sistema desarrollado en lenguaje Ruby, potenciado por Ruby on Rails y una base de datos PostgreSQL, cubriendo toda lo lógica del negocio.

Estudio Vienna

EstudioVienna es un proyecto para la automatización del proceso de arriendo de salas de grabación y ensayo. También permite gestionar la reserva de instrumentos o contratación de profesores. El sistema se encuentra desarrollado en lenguaje Ruby, potenciado por Ruby on Rails y una base de datos PostgreSQL. El sistema cuenta con sistema de pagos online (khipu) ya en beta, mejorando y facilitando la gestión del negocio.
